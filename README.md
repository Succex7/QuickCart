# QuickCart
QuickCart is a responsive e-commerce landing page built wit pure HTML, CSS and JavaScript. It's designed to adapt seamlessly to different screen sizes, offering an intuitive and visually appealing shopping experience for users. 

The page features a hamburger navigation menu on smaller screens for a clean and minimal layout while maintaining accessibility and responsiveness across devices.

## Features
- Built with pure HTML, CSS and JavaScript (no frameworks)
- Fully responsive design
- Hamburger menu for easy navigation on smaller screens
- Clean and modern UI/UX layout
- Lightweight and fast-loading page

## Technologies Used
- HTML5 - page structure and content
- CSS3 - styling and responsive layout
- JavaScript (Vanilla JS) - interactive hamburger menu
